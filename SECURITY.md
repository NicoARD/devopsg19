# Security Policy

## Reporting a Vulnerability
Please do not open public issues for security vulnerabilities.

Instead, report via:
- GitHub Security Advisory: https://github.com/NicoARD/devopsg19/security/advisories
- Or email the maintainers: security@devopsg19.net

We will acknowledge receipt within 3 business days and provide a timeline for a fix.

## Supported Versions
We generally support the current main branch and the latest tagged release.

## Guidelines
- Do not include sensitive data or credentials in reports.
- Provide minimal reproduction steps and environment details.
- If the issue involves SQL or data exposure, include sanitized examples.
